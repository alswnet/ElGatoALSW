# ElGatoALSW
Configuraciones para ElGato stream-deck de ALSW escrito en Python

# Instuciiones

* sudo apt update
* sudo apt upgrade
* sudo apt install python3-pip
* sudo apt-get install scrot python3-tk python3-dev
* pip3 install websockets
* pip3 install pyautogui
